# 前端专项算法

## 虚拟列表

虚拟列表用于优化长列表渲染性能。

**核心思想：**
- 只渲染可见区域
- 动态计算滚动位置
- 复用 DOM 节点

## LRU 缓存

LRU（Least Recently Used）缓存算法。

**实现要点：**
- 使用 Map 存储键值对
- 使用双向链表维护访问顺序
- 容量满时删除最久未使用的项

## 防抖和节流

### 防抖（Debounce）

在事件触发后等待一定时间再执行，如果期间再次触发则重新计时。

**应用场景：**
- 搜索框输入
- 窗口 resize

### 节流（Throttle）

在固定时间间隔内只执行一次。

**应用场景：**
- 滚动事件
- 鼠标移动

## 深拷贝

实现对象的深拷贝，处理循环引用。

## 事件委托

利用事件冒泡机制，将事件处理委托给父元素。
