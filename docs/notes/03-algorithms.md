# 核心算法思想

## 排序算法

### 快速排序（Quick Sort）

时间复杂度：平均 O(nlogn)，最坏 O(n²)
空间复杂度：O(logn)

### 归并排序（Merge Sort）

时间复杂度：O(nlogn)
空间复杂度：O(n)

### 堆排序（Heap Sort）

时间复杂度：O(nlogn)
空间复杂度：O(1)

## 动态规划（DP）

动态规划通过将问题分解为子问题来求解。

**核心思想：**
- 状态定义
- 状态转移方程
- 边界条件

## 贪心算法

贪心算法在每一步都做出当前最优的选择。

**适用场景：**
- 活动选择问题
- 最小生成树
- 最短路径问题
