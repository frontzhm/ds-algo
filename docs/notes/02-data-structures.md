# 基础数据结构

## 数组 & 链表

### 数组（Array）

数组是最基础的数据结构，支持随机访问，时间复杂度为 O(1)。

**特点：**
- 连续的内存空间
- 随机访问效率高
- 插入和删除需要移动元素，效率较低

### 链表（Linked List）

链表通过指针连接节点，支持动态扩容。

**特点：**
- 非连续的内存空间
- 插入和删除效率高
- 不支持随机访问

**示例代码：**

```typescript
// 单向链表实现示例
import { SinglyLinkedList } from '@ds-algo/core';

const list = new SinglyLinkedList<number>();
list.append(1);
list.append(2);
list.reverse();
console.log(list.head?.val); // 2
```

## 栈 & 队列

### 栈（Stack）

栈是一种后进先出（LIFO）的数据结构。

**应用场景：**
- 函数调用栈
- 表达式求值
- 括号匹配

### 队列（Queue）

队列是一种先进先出（FIFO）的数据结构。

**应用场景：**
- 任务调度
- 消息队列
- BFS 算法

## 哈希表

哈希表通过哈希函数将键映射到值，实现 O(1) 的平均时间复杂度。

**应用场景：**
- 快速查找
- 去重
- 缓存实现（LRU Cache）

## 树 & 堆

### 二叉树（Binary Tree）

二叉树是每个节点最多有两个子节点的树结构。

**常见类型：**
- 二叉搜索树（BST）
- 平衡二叉树（AVL）
- 红黑树

### 堆（Heap）

堆是一种特殊的完全二叉树，常用于实现优先队列。

**类型：**
- 大顶堆：父节点 >= 子节点
- 小顶堆：父节点 <= 子节点

**应用场景：**
- 优先队列
- Top K 问题
- 堆排序
