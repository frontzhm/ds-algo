前置准备 ​

学习目标 ​

在开始学习数据结构与算法之前，我们需要做好以下准备：

  1. 环境搭建

    • Node.js 18+ 和 TypeScript
    • 代码编辑器和调试工具
    • Git 版本控制

  2. 基础知识

    • JavaScript/TypeScript 基础
    • 基本的数据结构概念
    • 时间复杂度和空间复杂度

  3. 学习工具

    • LeetCode 账号
    • 代码测试框架（Vitest）

环境要求 ​

  • Node.js >= 18
  • pnpm >= 8
  • TypeScript >= 5.0
